## 📖 About The Project
A smart, console-based **Movie Recommendation System** built with **C#**. It analyzes user ratings and behavior to generate personalized, AI-driven movie recommendations using machine learning concepts, simulating platforms like Netflix.

## ✨ Key Features
* **User System:** Secure login, registration, and personalized watch history.
* **Explore Movies:** Browse, search (by genre, year, director), and rate movies (1-5 stars).
* **🧠 Smart AI Logic:** Implements intermediate AI algorithms:
  * *Content-Based Filtering:* Suggests movies based on similar genres, tags, and directors.
  * *Collaborative Filtering:* Compares users with similar tastes to recommend new movies (using Cosine Similarity).
* **Data Persistence:** Saves users, movies, and ratings locally using JSON.

## 💻 Tech Stack
* **Language & Framework:** C# / .NET
* **Machine Learning:** ML.NET
* **Design:** Clean Architecture, SOLID, and strict OOP principles.
